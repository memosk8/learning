Nuestra gama de proyectos web que le ofrece Mantisa:
Quizás usted se pregunte ¿qué diferencias existen?, a continuación le definimos cada proyecto web.

Página web

Una página web es información eléctronica estática, la cual puede contener texto, imágenes, sonidos, videos, enlaces, entre muchas más cosas. Generalmente las páginas web no proveen de muchas funcionalidades, aunque en algunas situaciones existen formularios. ¿Dónde se suelen usar las páginas web? en páginas web personales, blogs, galería o solo para compartir información.

Aplicación web

Una aplicación web, es un conjunto de distintas páginas web enlazadas dinámicas, las cuales su contenido aún esta sin determinar y este contenido se determinará justo después de que un usuario haya interactuado en ella. El objetivo principal de una aplicación web es proporcionar al usuario la posibilidad de realizar una o diversas tareas, además su costo es mayor y el tiempo de desarrollo también lo es, ¿dónde se suelen usar? en institutos gubernamentales, escuelas, bancos, empresas y entre muchas más.

Web móviles

En una web móvil se accede desde un dispositivo como las tabletas, telefonos móviles, y se va adaptando a cada resolución de pantalla, está optimizado para la navegación móvil. Se crean y se ejecutan desde un navegador con conexión wifi sin importar el sistema operativo que usted este usando.
